# MeasureApp - Landing Page

カメラで物の重さ・サイズをAI測定するiOSアプリ「MeasureApp」のランディングページです。

## 概要

このランディングページは、プログラミング初心者がAI（Claude）と一緒に7日間で開発したiOSアプリを紹介するために作成されました。

## 機能

- **重さ測定**: カメラで撮影して物の重量をAI推定
- **サイズ測定**: 物体のサイズを自動推定
- **カロリー計算**: 食べ物のカロリーと栄養情報を推定

## デザインコンセプト

- Kawaiiパステルカラー（ピンク・ブルー・イエロー）
- グラデーション効果
- アニメーション（星、雲などの装飾）
- レスポンシブデザイン

## デプロイ

このサイトはNetlifyで自動デプロイされます。

### ローカルでの確認

ブラウザで `index.html` を開くだけで確認できます。

### Netlifyへのデプロイ手順

1. GitHubにリポジトリを作成
2. Netlifyで「New site from Git」を選択
3. GitHubリポジトリを連携
4. デプロイ設定は自動で読み込まれます

## 技術スタック

- HTML5
- CSS3（カスタムプロパティ、グラデーション、アニメーション）
- Vanilla JavaScript（将来の拡張用）
